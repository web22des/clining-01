# 📁 Папка data

Здесь хранятся все данные для секций сайта (тексты, ссылки, настройки).

## 📂 Структура

```astro
📦data
 ┣ 📜README.md
 ┗ 📜site-config.ts # Глобальные настройки сайта

```

## ✏️ Как редактировать

1.  Откройте нужный файл (например, `sections/cards-data.ts`)
2.  Измените текстовые значения в кавычках
3.  **Не меняйте** структуру объекта и имена полей!

## ⚠️ Важно

- **Не удаляйте** поля из объектов (сломается вёрстка)
- **Не меняйте** пути к изображениям без проверки
- **Сохраняйте** тип данных (строка, число, массив)

## 🖼️ Изображения

Все картинки хранятся в `src/assets/`. В файлах данных указывайте **импорт**, а не строковый путь:

```ts
import img from "@assets/programming/01.jpg"; // ✅
{
    image: img;
} // ✅
```
